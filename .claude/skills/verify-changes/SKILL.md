# 验证代码变更

按层级验证代码修改是否正常工作，任一级别失败则停止并报告。

## 验证层级

### L1: 语法检查
- JavaScript: `node --check <file>`
- 检查是否有未定义引用、语法错误
- 参考: `references/checklist.md` 中的语言相关检查项

### L2: 脚本验证
- 运行项目的数据转换/构建脚本
- 确认输出文件已更新（时间戳、行数变化）
- 检查是否有错误输出

### L3: 运行时验证
- 启动应用（打开 HTML 文件或启动开发服务器）
- 验证修改涉及的功能正常工作
- 检查浏览器控制台是否有报错

### L4: 回归检查
- 快速冒烟测试核心流程
- 确认相关功能未受影响
- 如有回归，列出具体表现

## 输出格式

```
✅ L1 语法检查: 通过
✅ L2 脚本验证: node convert-dates.js 通过 (303 条)
✅ L3 运行时  : 页面正常渲染，功能 OK
⚠️ L4 回归    : 搜索功能偶发无结果（非本次修改引入）
```
